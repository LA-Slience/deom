# 电动车公共充电设施监管平台

## 项目简介

这是一个面向政府/行业监管级的电动车公共充电设施监管平台后台管理系统，核心目标是实现"全量接入、合规监管、安全预警、数据赋能"。

## 技术栈

- **前端框架**: React 18
- **构建工具**: Webpack 5
- **UI组件库**: Ant Design 5
- **图表库**: ECharts (echarts-for-react)
- **路由**: React Router 6
- **开发语言**: JavaScript (JSX)

## 功能模块

### 1. 领导驾驶舱与决策支持
- 全市充电态势总览（GIS地图展示）
- 热力图分析（分布、使用率、投诉）
- 收费合规监管看板
- 安全风险态势图

### 2. 基础档案与接入管理
- 运营商（企业）管理
- 充电站档案管理
- 充电桩设备台账
- 接口与接入配置

### 3. 收费监管与清算
- 价格政策管理（峰谷平尖电价）
- 收费数据核验与异常预警
- 对账与统计分析

### 4. 设备运行监控与运维
- 实时状态监控
- 故障报警中心
- 运维工单系统

### 5. 投诉举报与执法支撑
- 统一投诉受理（12345热线、APP、小程序）
- 执法取证支撑
- 信用评价体系

### 6. 安全监管与应急管理
- 消防安全监测
- 应急预案管理
- 应急事件处置

### 7. 系统设置
- 个人信息管理
- 密码修改
- 通知设置
- 用户管理
- 系统配置
- 安全设置

## 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm start
```

访问 http://localhost:12000 查看应用

### 构建生产版本

```bash
npm run build
```

## 项目结构

```
src/
├── components/     # 通用组件
├── layouts/        # 布局组件
│   └── MainLayout.jsx

│   ├── Safety/         # 安全监管
│   └── Settings/       # 系统设置

├── styles/         # 全局样式
├── utils/          # 工具函数
├── App.jsx         # 应用入口
└── index.js        # 渲染入口
```
